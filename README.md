# WeatherApp

Приложение для просмотра текущей погоды в выбранном городе.

## Функциональность

- Получение данных о погоде с помощью API OpenWeatherMap
- Возможность выбора города и сохранения его в истории запросов
- Отображение текущей температуры и погодных условий в выбранном городе
- Использование CoreData для хранения истории запросов
- Поиск города по названию


## Установка

1. Клонируйте репозиторий.
2. Откройте WeatherApp.xcodeproj в Xcode.
3. Соберите и запустите проект.
